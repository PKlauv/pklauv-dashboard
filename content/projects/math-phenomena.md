Math-Phenomena is a collection of interactive mathematical visualizations that bring abstract concepts to life in the browser. Each visualization lets you explore the parameters and see how they affect the output in real time.

## Visualizations

- **Lorenz Attractor** — the classic chaotic system, rendered as a 3D trajectory
- **Mandelbrot Set** — zoomable fractal explorer with color mapping
- **Möbius Strip** — 3D parametric surface you can rotate and inspect
- **Klein Bottle** — the non-orientable surface, rendered in 3D
- **Sierpinski Triangle** — recursive fractal with adjustable depth

## Built with

- Vanilla JavaScript and HTML5 Canvas for 2D renders
- Plotly.js for 3D surface plots
- Single-page app deployed on GitHub Pages
